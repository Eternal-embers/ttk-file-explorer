ttk file explorer

一款基于Python的标准TkGUI工具包的接口tkinter的美化库ttkbootstrap开发的轻量级的Windows文件浏览软件
GitHub下载 百度网盘下载 阿里云盘下载

对于很多Windows用户来说会是一个不错的选择

界面美观

采用现代化的界面设计，注重简洁性和美感。

极致绿色

运行响应快，无配置要求，最小资源使用。

易于使用

提供了高效且直观的文件浏览和管理功能。
