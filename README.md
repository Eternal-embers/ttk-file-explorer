##编写原因

本人基于tkinter的美化库ttkbootstrap开发了轻量级的Windows操作系统文件浏览软件——ttk文件浏览器，对于很多用户来说会是一个不错的选择。

当然,与其他的第三方文件管理软件相比可能有很多不足,但是它是为数不多的纯原生开发、无任何依赖tkinter软件,会持续优化,敬请期待！
