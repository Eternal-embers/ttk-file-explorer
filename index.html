使用原因

ttk file explorer 对于很多windows用户来说会是一个不错的选择。当然，与其他的第三方文件管理软件相比可能有很多不足，但是它是为数不多的纯原生开发、无任何依赖的 tkinter 软件，会持续优化，敬请期待！
